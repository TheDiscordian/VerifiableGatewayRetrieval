For verifying [CAR files](https://github.com/ipld/js-car) in a browser. Work in progress.

## Requirements

- [Python3](https://python.org)
- [npm](https://nodejs.org/en/download/)

## Setup

### Download the Source

```sh
git clone https://github.com/TheDiscordian/VerifiableGatewayRetrieval.git
cd VerifiableGatewayRetrieval
```

### Install libraries

```sh
npm install
npm run build
```

## Run Demo

Run `npm start` and navigate to http://localhost:8000/ in your browser or [click here](https://ipfs.io/ipfs/QmPGuqboSpcmRsXzUEgHo4jXkaSxFw7s5S3ESZGBwawFWg) to see a snapshot.
